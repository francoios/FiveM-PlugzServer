# What is life?

life est le script qui gerera la vie quotidienne du joueur:

* Sa faim
* Sa soif
* Son salaire de base (qui tombe toutes les minutes: 100$ pour l'instant)
* Position toute les minutes
* Skin du joueur
* Et tout ce qu'on pourra penser dans le genre
